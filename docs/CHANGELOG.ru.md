## [0.12.0] - 2026-03-04

### Добавлено
- **Переключатель источника зависимостей (yt-dlp/FFmpeg)** - Добавлен выбор источника в Настройки → Зависимости: бинарники приложения или системные бинарники
- **Подтверждение безопасности при переключении на системный источник** - Добавлено окно подтверждения при смене источника yt-dlp/FFmpeg на системный, чтобы избежать случайных изменений

### Изменено
- **Системная метка источника по ОС** - Метка системного источника теперь адаптируется по платформе (`Homebrew` на macOS, `PATH` на Windows, пакетный менеджер на Linux)
- **Автогенерация заметок релиза GitHub в build workflow** - В workflow публикации включен `generate_release_notes`, чтобы релизы публиковались с автоматически сгенерированными заметками
- **Интеграция кастомной строки заголовка в Windows** - Нативная строка заголовка Windows заменена на элементы управления в стиле приложения (область перетаскивания, свернуть/развернуть/закрыть)

### Исправлено
- **Запись истории загрузок в Windows** - Надежный захват итогового пути выходного файла в Windows, чтобы завершенные загрузки корректно попадали в историю библиотеки
- **Разбор пути при повторной загрузке в Windows** - Исправлено извлечение выходной папки для Windows-путей с разделителем `\`
- **Не-UTF-8 вывод yt-dlp в Windows** - Добавлен fallback-декодинг GBK/ANSI и обработка `--print-to-file`, чтобы корректно получать итоговый путь в локалях не UTF-8
- **Автообновление библиотеки после завершения загрузки** - История библиотеки автоматически обновляется, когда статус загрузки становится `finished`
- **Совместимость Douyin URL в modal-формате** - URL `douyin.com` с `modal_id` нормализуются в канонический формат `/video/{id}` в backend-вызовах yt-dlp и frontend-парсинге deep-link

## [0.11.1] - 2026-03-01

### Добавлено
- **Поддержка французского, португальского и русского языков** - Полная локализация интерфейса, настроек, сообщений об ошибках и меток метаданных на Français, Português и Русский
- **Локализация backend-ошибок** - Сообщения об ошибках backend (сбой загрузки, сетевые ошибки и т.д.) теперь переводятся на выбранный язык пользователя, а не всегда показываются на английском

### Изменено
- **Рефакторинг цепочки fallback для транскриптов** - Унифицирована логика fallback для AI Summary и Processing для более согласованного поведения

### Исправлено
- **Fallback транскриптов для Douyin и TikTok** - Улучшено извлечение транскриптов для видео Douyin и TikTok, которые ранее могли молча завершаться с ошибкой
- **Ошибки транскриптов и короткие подписи** - Ошибки транскриптов теперь сохраняются для диагностики, а короткие подписи принимаются как валидные
- **Параметры TikTok по умолчанию** - Настройки загрузки TikTok по умолчанию приведены в соответствие с платформенными ожиданиями

## [0.11.0] - 2026-02-20

### Добавлено
- **Браузерное расширение для загрузки в один клик (Chromium + Firefox)** - Можно отправлять текущую страницу видео из браузера в Youwee с выбором `Download now` или `Add to queue`
- **Настройка расширения в Settings** - Добавлен раздел Settings → Extension с прямыми кнопками загрузки и простыми шагами установки для Chromium и Firefox

### Изменено
- **Обновление UI/UX страниц YouTube и Universal** - Упрощены ввод ссылок, preview, очередь и взаимодействия в title bar для более чистого и единообразного интерфейса

### Исправлено
- **Единая логика разрешения зависимостей между функциями** - Унифицирована обработка источников yt-dlp/FFmpeg в download, metadata, channels и фоновом polling
- **Строгое поведение в режиме system** - Если выбран системный источник и бинарник отсутствует, приложение теперь явно показывает ошибку вместо тихого fallback

## [0.10.1] - 2026-02-15

### Добавлено
- **Настройки шрифта ASS** - Добавлена настройка семейства и размера шрифта субтитров для экспорта ASS и предпросмотра
- **Workflow переноса строки** - Добавлено быстрое авторазбиение строк и поддержка Shift+Enter при редактировании текста субтитров
- **Настраиваемый авто-повтор** - Добавлены настройки Auto Retry для загрузок YouTube и Universal с настраиваемым числом попыток и задержкой

### Изменено

### Исправлено
- **Диагностика сбоев загрузки** - Улучшены сообщения об ошибках yt-dlp с более понятными причинами, что помогает точнее обрабатывать временные ошибки и авто-повтор

## [0.10.0] - 2026-02-15

### Добавлено
- **Мастерская субтитров** - Добавлена страница "все в одном" для SRT/VTT/ASS с редактированием, инструментами тайминга, поиском/заменой, авто-исправлением и AI-действиями (Whisper, перевод, исправление грамматики)
- **Продвинутые инструменты субтитров** - Добавлены таймлайн waveform/spectrogram, синхронизация по смене сцен, realtime QC с профилями стиля, инструменты split/merge, режим переводчика (source/target) и batch/project операции
